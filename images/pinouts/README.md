# Images de Brochage

Placez ici les images de brochage des composants.

## Format recommandé
- Format: PNG ou JPG
- Taille: 800x600px recommandé
- Fond: Transparent ou blanc

## Noms de fichiers attendus
- `led-rouge.png` - LED Rouge 5mm
- `led-rgb.png` - LED RGB commune cathode
- (ajoutez d'autres images selon vos besoins)

## Comment créer les images
1. Dessinez le schéma du brochage (avec Paint, Photoshop, Illustrator, etc.)
2. Indiquez clairement chaque pin avec son nom
3. Sauvegardez au format PNG
4. Placez le fichier ici avec le bon nom
